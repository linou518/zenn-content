毎年14,000本以上のゲームがSteamに登録される。あなたのゲームは、なぜ見つけてもらえるのか？

多くのインディー開発者はSteamのレコメンドを「オカルト」と感じている——運がよければアルゴリズムに選ばれ、悪ければ埋もれると。だが実際には、Steamの可視度ロジックは非常に明確で、**予測・最適化が可能**だ。

本記事は2024年10月〜2025年7月の10,000本以上のゲームデータに基づき、Steamの発見アルゴリズムのコアメカニズムと、2026年における最高ROIのマーケティング手法を解説する。

---

## 1. Steam可視度の数式：オカルトではなく数学

Steamアルゴリズムの本質は以下の数式だ：

```
可視度スコア = (エンゲージメント率 × 総トラフィック) + コンバージョン品質

エンゲージメント率(40%) = (ウィッシュリスト + クリック + 動画再生) / 表示回数
トラフィック(30%)       = 全チャネルの総露出
コンバージョン品質(30%) = WL→購入率×0.4 + レビュースコア×0.3 + プレイ時間/同ジャンル平均×0.3
```

**エンゲージメント率の業界基準（2025年データ）：**

| エンゲージメント率 | 評価 |
|-------------------|------|
| 12%以上 | 優秀（アルゴリズムが積極的に推薦） |
| 8–11% | 良好（安定した露出） |
| 5–7% | 普通（加成ほぼなし） |
| 5%未満 | 危険（デス・スパイラル） |

重要な洞察：**アルゴリズムは良い結果を増幅し、悪い結果も増幅する**。エンゲージメント率が低い → 露出減 → データ悪化 → さらに露出減。多くの「リリース即死」ゲームは、発売前にすでに運命が決まっていた。

## 2. カバー画像（Capsule）：最もROIの高い投資

広告費に予算を使うより、$300でカバー画像のA/Bテストを行う方が圧倒的にROIが高い。

**データ：**
- 平均的なカバー画像のCTR：4–6%
- トップクラスのインディーゲーム：10–15%
- CTRが1%上昇 ≈ 約25%の追加オーガニック露出

**$300テスト法：**

1. 5パターン作成：ロゴ中心 / キャラクター中心 / アクション瞬間 / コアゲームプレイ展示 / 雰囲気重視
2. 各パターンに$60でSteam Spotlightを出稿
3. CTR最高のものを正式カバーに採用
4. 四半期ごとに再テスト

**実績：**
- Vampire Survivors：CTR 4.2%→9.7%（+131%）、オーガニック露出340%増
- Balatro：6.1%→11.8%、最終的に最も売れたDeck Builder

**高コンバージョンカバー公式：**

```
即座の認識（これは何？）
+ 視覚的インパクト（SteamのグレーBGで目立つ）
+ 感情的フック（好奇心を引く）
+ ジャンルシグナル（ターゲットプレイヤーが一目でわかる）
```

## 3. Tagストラテジー：三層構造でニッチを制す

**誤った戦略**：誰もが使う大流量Tagを積み重ねる。「Indie」は810万フォロワーがいるが、あなたは6,700位に埋もれる可能性がある。

**正しい戦略：三層Tag組み合わせ**

| 層 | 比率 | フォロワー目標 | 目標順位 | 役割 |
|----|------|--------------|---------|------|
| Foundation | 30% | 500K以上 | 追わない | 最低限の露出確保 |
| Competitive | 50% | 50K–300K | Top 50 | 安定した可視性 |
| Niche | 20% | 10K–50K | Top 10 | ニッチ市場を最大化 |

実例：あるインディータワーディフェンスが7つのTagでTop 20入り → 月8,000件の安定したオーガニックウィッシュリスト獲得。

## 4. ウィッシュリスト転換率：現実のデータ

10,656本のSteamゲームの実測データ（2024–2025年）：

- **初月の中央転換率：約27%**（ウィッシュリスト1万件 ≈ 初月2,700本販売）
- Early Access は正式リリースより約8%低い転換率
- 割引と転換率に**強い相関はない**
- 高価格ゲームは転換率がやや低いが、**ウィッシュリスト1件あたりの収益は高い**

**重要な閾値：**
- **7,000–10,000件**：Popular Upcoming入り（早期露出）
- **30,000件以上**：New & Trendingを狙える

## 5. 2026年マーケティングタイムライン

「ゲームが完成してからマーケティング」はもはや通用しない。2026年、Steam年間14,000本以上の中で、**遅れはそのまま死を意味する**。

2023年、「ローカルで大モデルを動かす」はオタクの極客プロジェクトでした。

2026年、Raspberry Pi（$35）は真の推論能力を持つ3Bパラメータモデルを実行でき、スマートフォン上の1Bモデルは毎秒2,500トークン以上を処理します。

これは「なんとか動く」ではありません。**エッジAIの質的転換点**です。

---

## はじめに：なぜ「小モデル」が今真剣に取り組む価値があるのか

**重要な転換**：高品質な学習データ + 知識蒸留により、10B未満のモデルが特定タスクで大モデルを**上回る**ようになっています。Phi-4-mini、Gemma 3、Qwen 2.5——これらの「小さな」モデルの推論・コード・数学ベンチマークは、2022年のGPT-4を既に超えています。

推論コストは**10〜100倍**低下。エッジ展開は「可能」から「推奨」へ。

## 2026年エッジモデル比較表

| モデル | パラメータ | 特徴 | 対象ハードウェア |
|-------|-----------|------|----------------|
| **Gemma 3 1B** | 1B | 2500+ tok/s、GSM8K 62.8% | スマホ/IoT |

## 三つの主要技術トレンド

### 1. MoE（Mixture of Experts）アーキテクチャのエッジへの浸透

Qwen3-30B-A3Bの例：総パラメータ30Bでも、MoEにより各推論でアクティブなのは約3Bのみ。

- 中級デバイスで3Bのスピードで30Bに近い知識密度を実現
- 「低スペックで大モデルを動かす」夢が現実に

### 2. 3Bが新スイートスポット

| パラメータ | 特徴 | 適合デバイス |
|-----------|------|------------|
| 1B | 超軽量、推論は制限あり | スマホ/IoT |
| **3B** | **真の推論、Piで動く** | **Raspberry Pi** |
| 7B | 大幅な能力向上 | 高性能エッジ |
| 10B+ | 実用的だが専用ハード要 | GPU搭載デバイス |

### 3. マルチモーダルの標準化

Qwen3.5-2BはPhi-4-mini並みで、**2Bパラメータでテキスト+画像をネイティブサポート**。

### 量子化比較

| 精度 | サイズ | 精度損失 | 用途 |


## 実践的な活用シナリオ

### シナリオ1：家庭/小チームサーバー

```bash
# Ollamaで始めるのが最も簡単
brew install ollama
ollama run llama3.2:3b
```

コスト：一回限りのハードウェア + 電気代。月額API費用ゼロ。

### シナリオ2：Agentクラスター最適化

- **複雑な推論** → クラウド大モデル（Claude/GPT-4）
- **分類・要約・フォーマット** → ローカル3B SLM

単純タスクは呼び出し量の60〜70%。置き換えでコストを大幅削減。

### シナリオ3：プライバシー重視アプリ

医療・法務・金融データ：**ローカルモデル優先**。コンプライアンスコスト > API節約。

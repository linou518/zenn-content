OCMのコマンドラインインターフェースが完成。`ocm node list`、`ocm node add`、`ocm node remove`、`ocm backup`、`ocm restore`、`ocm status`——すべての管理操作がCLIから実行可能に。対話型プロンプト、カラー出力、進捗表示を実装。WebUIとCLIの二刀流で、GUIが好きな人もターミナル派も満足する管理体験を提供。

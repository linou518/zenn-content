## はじめに：表面利回り9%の物件、手残りはいくら？

物件情報サイトでよく見かける「表面利回り9.3%！」——この数字を見て購入した場合、実際の月間手残りはいくらでしょうか？

**答えは月16万円程度が現実解。しかも返済比率55%でギリギリです。**

「表面利回りが良く見えたから買った」という判断が、不動産投資で最も多い失敗パターンです。本記事では4ステップのCF計算法と1億円・2億円クラスのリアルな試算例で、正しい投資判断の方法を解説します。

---

## 1. 表面利回りと実質利回りの違い

**例：年間家賃収入576万円・経費100万円・物件価格6,200万円**

| | 計算 | 結果 |
|--|------|------|
| 表面利回り | 576 ÷ 6,200 × 100 | 9.3% |
| **実質利回り** | **(576 − 100) ÷ 6,200 × 100** | **7.7%** |

1.6%の差が20年間で積み重なると、数百万円単位の影響になります。

---

## 2. 4ステップCF計算法

### Step 1：収入の設定

| 築年数 | 現実的な入居率 |
|--------|-------------|
| 新築〜5年 | 95〜100% |
| 築10年 | 90%前後 |
| 築20年以上 | 80%以下 |

「95%固定」ではなく経年劣化を折り込むことが鉄則。

### Step 2：経費の見積もり（家賃収入の15〜20%が目安）

- 管理委託料：3〜5%
- 修繕・清掃費：3〜5%
- 固定資産税・保険料：年数十万円

### Step 3：ローン返済の計算

**返済比率 = 年間返済額 ÷ 年間家賃収入 → 50%以下を死守**

変動金利の場合は+1%のストレステスト必須。

### Step 4：CF算出

```
年間CF = 有効年収 − 経費 − 年間返済額
```

---

## 3. リアル試算例

### 1億円クラス（10戸、家賃6万円/戸）

| 項目 | 金額 |
|------|------|
| 満室時年収 | 720万円 |
| 空室控除10% | −72万円 |
| 経費15% | −97万円 |
| ローン返済（8,000万/2%/30年） | −355万円 |
| **月額CF** | **約16万円** |
| 返済比率 | 55%（注意） |

### 2億円クラス（20戸、家賃6万円/戸）

| 項目 | 金額 |
|------|------|
| 有効年収（空室10%） | 1,296万円 |
| 経費・ローン | −904万円 |
| **月額CF** | **約33万円** |

---

## 4. やっておくべきストレステスト

| シナリオ | 内容 | 確認点 |
|---------|------|--------|
| 空室悪化 | 入居率80%で再試算 | CFがマイナスになるか |
| 金利上昇 | +1%で返済額を再計算 | 返済比率60%超えるか |
| 修繕集中 | 築15年目に300万支出 | 手持ち資金で対応可能か |

3パターン全てでCFプラスなら「守りの硬い物件」と判断できます。

---

## まとめ

月16万円の手残りは地味です。でも30年後に借入ゼロで資産1億円が残るなら、それは立派な資産形成戦略。

賃貸経営の判断軸は**「表面利回り → 実質利回り → CF → ストレステスト」の4段階深掘り**にあります。数字で損を防ぐことが、長期的な不動産投資成功の本質です。
